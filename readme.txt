== Description ==
= HOW TO USE CUSTOM CSS ELEMENTOR
it’s a plug-and-play tool.  After installing the \"Custom CSS Option For Elementor\" plugin, you won\'t find a dedicated settings page. Instead, follow these steps to access the CSS customization options. Begin by navigating to the editing panel of any Elementor Widget. Once there, locate the Advanced tab section and click on it. Within this section, you will notice a new accordion called \"Custom CSS Option For Elementor\". Expand the accordion to reveal a field where you can input CSS codes for your widgets.

Direction: Edit any Elementor Widget > Advanced Tab > Custom CSS Option For Elementor

== Installation ==
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the \'Plugins\' menu in WordPress.

== Frequently Asked Questions ==
**How do I install \"Custom CSS Option For Elementor\"?**
To install the free version of \"Custom CSS Elementor\", follow the steps below:
From your WordPress dashboard -> Go to Plugins -> Click on \'Add new\'-> In the Search field, enter \"Custom CSS Option For Elementor\" and choose \"Custom CSS Option For Elementor\".
Press Install -> After installation, click Activate.

**Will it work with any other Elementor addons?**
Yes, this plugin will work with any other add-ons or Elementor.

**Do I need to install Elementor  PRO for this plugin?**
No! you don\'t need Elementor PRO to use this plugin. 

**Will this plugin automatically work after activation?**
Yes. You don’t need to do anything else other than activate the plugin.

**Is there any security thread after activating this plugin?**
No. This plugin is highly secure. It will sanitize user input so that the user can’t face any scripting attack.
